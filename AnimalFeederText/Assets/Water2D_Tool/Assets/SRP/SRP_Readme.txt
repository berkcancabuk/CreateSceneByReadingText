To make Water 2D Tool compatible with Unity LWRP/URP, follow the instructions bellow:

1. In the package manager, make sure that you have the latest version of LWRP/URP instaled.
2. Right click on the Assets folder and select Import Package/Custom Package. In the window that pops up, go to where your project is and
   from the Water2D_Tool/Assets/SRP folder import Water2DTool_LWRP.unitypackage if you are using Unity 2019.1 or 2019.2 and 
   Water2DTool_URP.unitypackage if you are using Unity 2019.3 or newer.
